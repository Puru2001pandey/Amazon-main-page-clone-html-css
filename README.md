# 📦 Amazon Homepage Clone

A pixel-perfect, responsive clone of the Amazon homepage built from scratch using clean, semantic HTML5 and custom vanilla CSS3. 

This project demonstrates strong foundational skills in layout design, CSS Flexbox/Grid, and replication of complex commercial user interfaces.


## ✨ Features

- **Pixel-Perfect Replication**: Recreated the iconic Amazon header (search bar, nav items, country selector, cart icon).
- **Responsive Card Layout**: Grid sections featuring multiple product category boxes with clean layout styling.
- **Hero Banner**: Fully styled hero showcase mimicking the promotional slider area.
- **Vibrant CSS Effects**: Smooth hover states, transition animations, and custom scrollbars to match the original experience.
- **Semantic HTML**: Structurally built for SEO best practices and accessibility.

## 🛠️ Tech Stack

* **HTML5**: Semantic markup for layout and structure.
* **CSS3**: Custom styling, Flexbox, CSS Grid, animations, and variables.

## 🚀 Getting Started

### Prerequisites

To run this project locally, all you need is a modern web browser.

### Installation & Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Puru2001pandey/Amazon-main-page-clone-html-css.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Amazon-main-page-clone-html-css
   ```
3. Open `index.html` in your favorite browser:
   * Double-click `index.html` in your file explorer, OR
   * Run it via terminal:
     ```bash
     start index.html  # On Windows
     open index.html   # On macOS
     ```

## 📐 Design Architecture

* **Layouts**: Heavy utilization of `Flexbox` for alignment in the top navigation bar and footer.
* **Responsive Grid**: Recreated the product grid sections using `CSS Grid` and flex containers.
* **Colors**: Hardcoded Amazon brand palettes (e.g., `#131921` for navbar, `#febd69` for highlights).
